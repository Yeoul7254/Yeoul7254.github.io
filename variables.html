파이썬 변수와 자료형

파이썬에서 변수는 데이터를 담는 이름 붙은 저장 공간입니다. 변수에 값을 할당할 때 자동으로 자료형이 결정됩니다.

주요 자료형

자료형 설명 예제
int 정수형 (예: 1, 42, -5) x = 10
float 실수형 (예: 3.14, -0.001) pi = 3.14
str 문자열 (예: "안녕하세요", 'Python') name = "Alice"
bool 불린형 (참 또는 거짓, True/False) flag = True
list 리스트 (여러 값을 순서대로 저장) nums = [1, 2, 3]
dict 딕셔너리 (키-값 쌍 저장) info = {"age": 30, "city": "Seoul"}

간단한 예제

# 변수 선언과 출력
x = 5
y = 2.5
name = "홍길동"
is_student = True

print("x =", x)
print("y =", y)
print("name =", name)
print("학생인가요?", is_student)

위 코드를 실행하면 아래와 같은 결과가 나옵니다:

x = 5
y = 2.5
name = 홍길동
학생인가요? True
